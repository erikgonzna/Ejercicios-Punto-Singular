Para la correcta ejecución del programa se deberá:

1. Ubicarse en la clase PrincipalMorse del Proyecto y ejecutarse

2. Ingresar por consola un codigo morse, separando cada símbolo por un espacio, o ingresar un texto escrito con letras mayúsculas
y números.

3. Preionar enter, y aparecerá el mensaje descifrado o cifrado, según sea el caso.
